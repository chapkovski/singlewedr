Dictionary: At the top of the screen, you will see a dictionary that shows the correspondence between each letter of the alphabet and a specific emoji. For example, if 😊 corresponds to 'A', every time you see 😊, it represents the letter 'A'.
Encoded Word: Below the dictionary, you will see a word that is encoded using these emojis. For instance, if you see 😊😎👍, and 😊 corresponds to 'A', 😎 to 'B', and 👍 to 'C', the encoded word is 'ABC'.
Input Field: You will also see an input field where you can type in the decoded word.
